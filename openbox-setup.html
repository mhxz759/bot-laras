Openbox Setup

Here are the setup instructions for Openbox
  1. Open up an Internet browser.
  2. Now you need to enter the M3U URL on it.
  3. You will be asked to save a file.
  4. Click on Save as and add any name.
  5. Now you can copy the file to a USB flash drive. Then you need to connect the USB flash drive to Openbox.
  6. Now you need to take the remote controller and follow these steps.
  1. Go to menu
  2. Go to Network local settings
  3. Go to Network App
  4. Go to IPTV Palace
  5. Press on Yellow Button to read from the USB drive.
  6. Now you can see the command saying success on Openbox.

If you still have any issues, simply restart Wi-Fi router and Openbox
